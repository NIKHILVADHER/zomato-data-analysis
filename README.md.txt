# Zomato Restaurant Data Analysis

## Overview
This project analyzes restaurant data from Bengaluru to understand trends in ratings, pricing, restaurant types, and online ordering.

## Tools Used
Python
Pandas
Matplotlib
Jupyter Notebook


## Key Insights
- Most restaurants have ratings between 3.5 and 4
- Around 65% of restaurants offer online ordering
- Only about 15% allow table booking
- Restaurants with table booking have higher ratings
- Average cost for two people is around ₹750
-Top 10 restaurants chains in Bengaluru

## Dataset
The dataset used in this project is available here:

https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants

Download the dataset and place `zomato.csv` in the same folder as the notebook before running the project.

Zomato Restaurant Dataset (Bengaluru)

## Conclusion
Bengaluru's restaurant industry is dominated by mid-range restaurants with strong adoption of online ordering platforms.
